Class RMIManager

The core class for handling RMI connections. This can be used to invoke remote functions or expose functions to the remote.

NOTE: As WebSockets are bidirectional, "client" and "remote" are arbitrary. A browser can be both client and remote if it both calls functions on a server while also exposing functions to the server.
