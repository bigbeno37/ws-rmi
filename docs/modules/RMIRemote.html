Const handleRequest

  • handleRequest(rmi: RMIRequest, functions: {}): Promise<RMIRemoteResponse>
    • Defined in RMIRemote.ts:20

    Handles an {@link RMIRequest} and produces either an {@link RMIRemoteResult} if the specified target:

    1. Exists
    2. Is a function
    3. Doesn't throw an Error

    Otherwise, an {@link RMIRemoteError} is produced detailing that an error occurred. For security purposes, a generic error message is generated. If an Error is thrown, it is propagated back to exposeFunctions where a given error handler can handle it further (e.g. console logs, output to a log file, etc.)
